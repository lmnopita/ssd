# Footer credential badges

Drop badge PNGs here (transparent, ~96px tall) as you earn them, then uncomment the
`<ul class="footer-badges">` block in each page's footer. Suggested filenames:

- fear-free.png                 Fear Free Certified Professional
- red-cross-pet-first-aid.png   Red Cross Pet First Aid & CPR
- force-free.png                Force-Free

Note: on 404.html use root-relative paths (/images/badges/…); all other pages use ./images/badges/…
